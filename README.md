# 🏆 Desafios: Clean Code, Big-O e SOLID

Bem-vindo ao laboratório prático de Engenharia de Software. Aqui você aplicará os conceitos aprendidos nas aulas 10 e 11.

## 📁 Estrutura dos Desafios

1.  **[01-complexidade-ciclomatica](./01-complexidade-ciclomatica):** Refatore uma função com excesso de `ifs` usando Table-Driven Design.
2.  **[02-big-o](./02-big-o):** Otimize uma busca de `O(n²)` para `O(n)`.
3.  **[03-solid](./03-solid):** Quebre um script monolítico aplicando SRP e DIP.

## 🚀 Como Executar

Cada pasta possui seu próprio `package.json`. Para rodar os desafios:

```bash
cd 01-complexidade-ciclomatica # ou outro desafio
npm install
npm test
```

## 🤖 Uso da IA

A IA está configurada neste diretório para atuar como **Mentora**. Ela **não** alterará seus arquivos nem dará a resposta pronta. Use-a para entender conceitos e tirar dúvidas sobre os erros dos testes.
